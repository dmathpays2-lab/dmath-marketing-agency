# Business Repository Structure

Organized by business vertical for easy transfer and management.

## 📁 Folder Structure

```
businesses/
├── mca/                          # MCA Lead Generation Business
│   ├── leads/                    # Lead lists, customer data
│   ├── templates/                # Email templates, scripts
│   ├── docs/                     # Business documentation
│   └── scripts/                  # Automation scripts
│
├── think-plus-energy/            # Energy Broker Business
│   ├── research/                 # Company research, market data
│   ├── marketing/                # Marketing strategies, campaigns
│   ├── compliance/               # Legal, PUCT rules, restrictions
│   └── strategies/               # Go-to-market plans
│
├── momentum-tech/                # Tech/Development Projects
│   ├── projects/                 # Active development
│   ├── docs/                     # Technical docs
│   └── notes/                    # Meeting notes, ideas
│
└── shared/                       # Cross-business resources
    ├── tools/                    # Tools used across businesses
    ├── templates/                # Reusable templates
    └── systems/                  # Agent systems, workflows
```

## 🔄 Transfer Instructions

To move to a new system:
1. Zip the entire `businesses/` folder
2. Transfer to new workspace
3. Unzip and resume operations
