## Kevin Jeppesen QMD Implementation

### What We Learned
Kevin Jeppesen's "trick" is the **combination of three systems**:
1. **QMD** (Query-Memory-Document) - Hybrid search backend
2. **Memory Flush Before Compaction** - Saves important details before context window resets
3. **Self-Improving Agent** - Agent that gets better over time

### Kevin's Key Insight (from video @ 43:16)
> "Memory flush kicks in right before compaction and says: 'Wait, before you throw away older messages, let me write down anything important first.'"

Without this, important details get lost during context compaction.
