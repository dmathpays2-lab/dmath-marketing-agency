## Notes

This project started after watching Nat Eliason's video about his bot Felix making $14,718 in 3 weeks. The key insight was the 3-layer memory system:

1. **Knowledge Graph** (PARA) - Structured long-term storage
2. **Daily Notes** - Raw daily activity logs
3. **Tacit Knowledge** - Preferences, habits, lessons learned

The smart cron system runs consolidation scripts so the AI doesn't timeout.
