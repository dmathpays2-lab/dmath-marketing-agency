## OpenClaw Platform Research

**Time:** 02:08 GMT+8

### What is OpenClaw?

- **Open-source AI agent framework**
- **Created by:** Peter Steinberger (founder of PSPDFKit)
- **Released:** Late 2025
- **GitHub stars:** 180,000-200,000+ by Feb 2026
- **Core concept:** Turns your computer into an autonomous employee

### Key Capabilities

**1. Proactive Agents**
- Unlike ChatGPT (waits for prompts), OpenClaw acts autonomously
- Runs 24/7 on your hardware
- Spawns sub-agents for specialized work
- Completes tasks while you sleep

**2. Tool Integration**
- Executes shell commands
- Edits files
- Browses the web
- Controls browser
- Manages calendars/email
- Sends messages (WhatsApp, Telegram, Slack, iMessage)
- Git operations (commit, merge, PR)
- Deploys applications

**3. Memory System**
- Markdown text files in Finder
- Long-term context retention
- Searchable memory
- Individual memory files per agent

**4. Communication Channels**
- WhatsApp
- Telegram
- Discord
- Slack
- iMessage
- SMS (via integrations)
- Voice (via Twilio + ChatGPT Voice API)

**5. Automation Features**
- **Cron jobs** - Scheduled tasks
- **Heartbeat polling** - Periodic checks
- **Sub-agent spawning** - Delegate to specialized agents
- **Skill system** - Download/install new capabilities

**6. Local-First**
- Runs entirely on your hardware
- Self-hosted
- Data stays local
- API calls only to LLM providers (Claude, GPT, DeepSeek)
- Can run local models (GLM 470, etc.)

### The MolT Stack

OpenClaw is part of a broader ecosystem:

**1. OpenClaw** (Core)
- Self-hosted autonomous agent
- Executes tasks locally

**2. MolTBook** (Social Network for AIs)

**3. MolTHub** (Skills Marketplace)
- Community-built skills/extensions

**Components:**
1. **Agents** - Role-based AI operators with identity, permissions, workspace
